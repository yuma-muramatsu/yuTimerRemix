# Work Timer - 集中と休憩のビジュアルタイマー

Time Timer風の視覚的な残時間表示と、タスク管理機能、インターバル機能を備えたワークタイマーです。

## 🌟 主な機能

- **ビジュアルタイマー**: 残り時間を扇形の図形で表示。直感的に進捗がわかります。
- **タスク管理**: これから取り組むタスクを入力し、履歴に記録可能。
- **インターバル機能**: 集中（ワーク）と休憩を自動で繰り返します。
- **カスタマイズ**: プリセット時間（25分、15分、10分、5分）のほか、カスタム時間設定も可能。
- **サウンド通知**: ワーク完了や休憩終了時に心地よいチャイムが鳴ります。
- **永続化**: ブラウザの `localStorage` を使用して、タスク履歴を保存します。

## 🚀 使い方

1.  **タスクを入力**: 画面上部の入力欄にタスク名を入力します。
2.  **時間設定**: ショートカットボタンまたは数値を入力して「設定」をクリックします。
3.  **インターバルの設定（任意）**: 自動で休憩を繰り返したい場合は「インターバル」をONにします。
4.  **スタート**: 再生ボタンをクリックして開始！
